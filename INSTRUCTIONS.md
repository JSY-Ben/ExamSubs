# ExamSubs Instructions

This guide explains how to use ExamSubs from both staff and student perspectives.

## Staff Workflow

### Create an exam
1. Go to the staff portal and select "Create Exam".
2. Fill in Exam ID, Exam Title, and the submission window (start/end date and time).
3. Set pre/post buffer minutes if needed.
4. Add Required Documents (these are the student upload slots).
5. Optionally configure naming templates for stored files/folders.
6. Optionally add Exam Materials for students to download.
7. Optionally set an exam access password.
8. Save the exam.

### Add or manage a student roster (optional)
1. Open the exam and click "Student Roster".
2. Enable the roster and choose identification mode:
   - Menu: students pick their name from a list.
   - Password: each student gets a unique password.
3. Add students manually or import a CSV.
4. Save the roster.
5. If using passwords, print student cards from the roster page.

### Manage exams
- Edit exam details, documents, materials, or passwords as needed.
- Mark exams completed or reopen them from the exam page.
- Download all submissions or individual files.
- Reset a submission to allow a student to submit again (previous files are archived).

## Student Workflow

### Access an exam
1. Open the student portal.
2. Select the exam and click "Submit Files".
3. If prompted, enter the exam access password and/or your student password.

### Download exam materials
1. On the student portal, use "Download Exam Materials" on the exam card.
2. Download a single file or all materials as a ZIP.
3. If a password is required, you will be prompted first.

### Submit files
1. Enter your name and candidate number (or select your name if roster mode is enabled).
2. Upload the required documents.
3. Confirm the final submission checkbox.
4. Click "Submit Files".

### Replace a submission
- If a submission already exists, the portal will warn you.
- Confirm the replacement checkbox to submit again.

### Cancel a submission
- Use "Cancel submission" to return to the exam list.
  - This also clears your roster password session, if used.
